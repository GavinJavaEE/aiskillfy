AISkillfy 是一个AI Agent技能生态平台，专注于提供标准化、可共享、可直接调用的Prompt Skill，搭配强大的嵌入式AI助手，打造「技能商城+文档知识库+个人Agent空间」的完整闭环，助力开发者快速落地AI Agent应用，降低技能开发与使用门槛。

✨ 核心功能
- 智能AI助手（核心）：内置全站嵌入式AI对话助手，支持两大核心能力——① 对话中智能检索站内文档、Skill资源，以引用形式精准推送；② 支持个人Agent空间装载Prompt Skill，装载后可在对话中直接调用，实现技能联动。
- 标准化Skill商城：收录高质量Prompt Skill，按多类场景清晰划分，覆盖开发工具、数据分析、AI助手、自动化、设计工具等多个领域，所有Skill遵循自研标准化格式，可直接下载、装载使用。
        

        🔗 技能商城链接：`[Skill Market](https://www.aiskillfy.com/skills)`
      
- 个人Agent空间：为每位用户提供专属Agent管理空间，可自由装载、启停、管理各类Skill，实现个性化技能组合，适配不同使用场景。
- 完整文档知识库：包含多篇深度技术文档，涵盖AI Agent技术详解、Skill使用指南、自研协议规范等内容，助力用户快速上手、开发者深入学习。


        🔗 文档中心链接：`[Document Center](https://www.aiskillfy.com/documents)`
      
- 开放协议规范：自研 `skill.md` 标准化格式，打造开放可共享的AI Agent Skill协议，支持跨平台兼容，助力开发者共建技能生态。
📊 Skill分类（持续更新）


- Dev-tools：Docker配置生成、数据库表结构设计、API调试与文档生成等开发相关技能
- Data-analysis：用户画像生成、反馈分析、SWOT分析等数据处理技能
- AI-assistant：多语言翻译、内容润色、竞品分析等智能助手类技能
- Automation：浏览器自动化、脚本执行等自动化操作技能
- Design-tools：UI/UX设计、直播脚本撰写、短视频文案生成等设计类技能
- Others：各类通用型补充技能
